# Cyacle 对象管理 (Object Management) 规范

此 Skill 用于指导 AI 在 Cyacle 项目中实现数据卡片、详情抽屉和业务表格。

## 01. 数据卡片 (Data Card System)

| 模式 | 结构要求 | 适用场景 |
| :--- | :--- | :--- |
| **极简模式** | 仅名称 | 简单列表选择 |
| **标准模式** | 图标标签 + 名称 | 需要区分类型的列表 |
| **详情模式** | 名称 + 关键数值/单位 | 计算结果、知识库列表 |
| **综合模式** | 头部(Icon+Name) + 折叠/展开正文 | 复杂对象，列表直接查看详情 |

### 交互规则
- **悬停状态**: 背景切换为 `bg-[#EDF7FA]`，右侧浮现操作按钮组（替换、删除）。
- **删除状态**: 容器透明度 50%，文字带中划线，图标切换为垃圾桶。

## 02. 对象抽屉 (Object Drawer)

### 物理特性
- **定位**: 位于内容区内部右侧（非阻塞式 `InnerDrawer`）。
- **交互**: 无遮罩，点击外部自动关闭，支持独立滚动。
- **结构**: 固定标题栏 + 滚动内容区 + 固定底部操作栏。

## 03. 对象表格 (Object Table)

- **行高**: 严格固定为 `36px` 或 `48px`。
- **斑马纹**: 奇数行 `bg-white`，偶数行 `bg-gray-50`。
- **表头样式**: 背景色 `#EDF7FA`，文字 `14px`。
- **操作列**: 固定在最右侧。
